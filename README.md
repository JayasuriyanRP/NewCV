# Resume Web Application

A modern, interactive resume web app built with React and Tailwind CSS. Easily customizable, featuring sections for experience, education, skills, certifications, and more.

## Features

- Dynamic resume sections (Experience, Education, Skills, Certifications, Projects, Hobbies)
- Print-friendly layout
- Easy data updates via JSON files
- Responsive design for all devices

## Getting Started

### Prerequisites

- Node.js (v16 or above)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-resume-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd NewCV
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

```bash
npm run dev
```

Open your browser and go to `http://localhost:5173`.

### Build for Production

```bash
npm run build
```

## Folder Structure

- `src/` - Main application source code
- `public/` - JSON data files and assets
- `images/` - Profile and project images
- `featured/` - Additional featured content

## Customization

- Update JSON files in `public/` to change resume content
- Modify components in `src/components/` for layout and styling
